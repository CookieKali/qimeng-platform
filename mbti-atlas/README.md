# MBTI 关系谱 · 十六型人格互动图鉴

动漫风格的单页 HTML 应用，展示 16 种 MBTI 人格的关系网络、优缺点、恋爱风格与伴侣匹配。

## 功能

- **关系星图**：D3 力导向交互图，金/青/红连线分别表示最佳伴侣、默契拍档、磨合组合
- **详情侧栏**：点击节点查看完整档案（动漫 archetype、优缺点、恋爱需求、职业、成长建议等）
- **人格卡片**：16 张可点击卡片快速浏览
- **兼容矩阵**：16×16 热力式对照表
- **配对测算**：任意两种类型的兼容指数估算
- **视觉**：星空渐变、樱花粒子、玻璃拟态、自定义 SVG 头像

## 使用

直接用浏览器打开 `index.html` 即可，无需构建或服务器。

```bash
open index.html
# 或
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```
