# 企盟小程序 · 使用说明

## 1. 本地运行（模拟器）

1. 仓库根目录执行：`./scripts/start_backend.sh`
2. 微信开发者工具打开 **`qimeng-miniprogram/`**（勿打开上一级目录）
3. 详情 → 本地设置 → 调试基础库 **3.4.10+** → 勾选 **不校验合法域名**
4. 点击 **编译**（日常开发不要用「预览」上传包）
5. 登录：`18800000001` / `123456`（见下方账号表）

模拟器 API：`http://127.0.0.1:8000`

## 4. 常用功能入口

| 功能 | 入口 |
|------|------|
| 任务 / 活动 | 任务&活动 Tab（活动为子分段） |
| 发布任务 / 活动 | 任务&活动页右下角 |
| 空间站 / 场地预约 | 任务&活动右上角「空间站」或 我的 → 空间站预约 |
| 空间站枢纽（股东/结算） | 空间预约页 →「空间站」Tab → 点卡片 |
| 分润 / 引荐 | 分润 Tab；`pending` 流水可点确认 |
| 贡献积分 | 分润页入口 或 我的 → 贡献积分中心 |
| 我的记录（任务/活动详情） | 我的 → 记录 → 点击卡片 |
| 活动圈 | 通讯录 → **活动圈**（同场报名且非好友） |
| 收件箱 / 私信 | 通讯录顶部消息入口 |
| 超管审预约 | `19900000000` 登录 → 我的 → 预约审核 |

## 5. 联调示例

### 好友（甲 / 乙）

1. 甲登录 → 通讯录 → 找乙 → 互换名片 / 加好友  
2. 乙登录 → 顶部好友申请 → 接受  
3. 甲再登录 → 好友 Tab 可见乙  

### 活动圈

1. 甲、乙 **分别登录**，对 **同一场活动** 报名（任务&活动 → 活动）  
2. 甲 → 通讯录 → **活动圈** → 应出现乙 → **申请加好友**  

### 任务闭环

甲发布任务 → 乙报名 → 甲录用 → 乙提交交付 → 甲验收（任务&活动 → 我发布 / 我承接）

## 6. 真机调试

手机不能访问 `127.0.0.1`，需 Mac 局域网 IP：

```bash
./scripts/sync_api_host.sh
./scripts/start_backend.sh
```

- 手机与 Mac **同一 WiFi**  
- 开发者工具勾选 **不校验合法域名**  
- 手机浏览器访问 `http://<Mac IP>:8000/api/health` 应有 JSON  

登录页会显示当前 API 地址（来自 `config.local.js`）。

## 7. 常见问题

| 现象 | 处理 |
|------|------|
| 预览上传失败 | 用 **编译** + 模拟器，或检查网络/代理 |
| 白屏 | 清缓存、删 Storage 里 `qm_token`、重新编译 |
| 真机连不上 API | `sync_api_host.sh`、重启后端、查防火墙 |
| Console `[object Object]` 且仅预览失败 | 微信上传包问题，非业务 API |
